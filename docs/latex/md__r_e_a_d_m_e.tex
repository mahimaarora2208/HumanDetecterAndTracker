In this project, a Human Detector and Tracker has been developed. To achieve this, we used the pre-\/trained model of Yolo-\/\+V5. The last layer of the \mbox{\hyperlink{class_yolo}{Yolo}} neural network was frozen so that it only detects humans in the image. After detecting the humans in the image there distance was calculated from the camera. Assumption we have taken is that the human in average is of 5ft.

Four classes were implemented to achieve this objective, in which the master class is {\bfseries{Human-\/\+Detector}}.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{Team\+:\+Group 4}\label{md__r_e_a_d_m_e_autotoc_md2}
